Cards Introduction

Congratulations, you've just installed and built Cards. It wasn't that hard right? Cards is a little, opiniated and pretty simplistic content-management-system. The default theme is inspired by Google's Material-Design guidelines with some little changes here and there.

History

Cards started as my own personal homepage. It featured different categories, a simple to use tag parser and a pretty looking design. I quickly thought about making it accessible to anyone. In this version however, i cut out a lot of features that i felt we're unnecessary for the average user.

For example, i removed the support for my own tag parser, which i hadn't updated since September 2015. I also removed a lot of stuff that i built for myself.

Build

Just launch the setup.php file via the browser or command-line. After everything finished, it will disable itself.

How to add a new category

Create a new folder inside the storage/articles folder and name it after your category.

How to add a new article

Inside the newly created category folder, create a new folder for your article. PHP sorts these articles based on alphanumeric sorting done by the filesystem, so you should use names similar to this format: 2016_03_16_sometitle.

Next, place a file called article.html inside the newly created directory. This will is a standard html file where you can write your article.

If you want to embed images, videos or other file-system related stuff, use the following syntax: %%THIS%%/image.jpg for the src path.

Inside this file, a div represents a card. You can have multiple cards inside one article. Play around with different tags to see how they are being formatted.
